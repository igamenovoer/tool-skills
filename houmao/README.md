# Houmao Skills

Houmao system skills mirrored from:

```text
igamenovoer/houmao/src/houmao/agents/assets/system_skills/
```

Install with the Skills CLI:

```bash
npx skills add igamenovoer/tool-skills/houmao
```

To install all skills for Codex non-interactively:

```bash
npx skills add igamenovoer/tool-skills/houmao -g -a codex --all -y
```
